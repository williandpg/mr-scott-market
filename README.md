<h1><strong>Mr. Scott Market</strong></h1>

<h2><strong>Descrição</strong></h2>
<p align="justify">
  Este prokjeto implementa uma aplicação que registra os itens vendidos em uma tabela e manipula esses dados com TypeScript.
  O código foi pensado para ser extensível, pois no futuro o mercadinho poderá incluir novos tipos de itens além de comidas e bebidas.
</p>

<h2><strong>Tecnologias Utilizadas</strong></h2>
<ul>
  <li align="justify">
    <a href="https://www.typescriptlang.org/" target="_blank"><strong>TypeScript</strong></a>.
    Tipagem estática e recursos de generics para escalar a modelagem de itens.
  </li>
  <li align="justify">
    <a href="https://nodejs.org/" target="_blank"><strong>Node.js</strong></a>.
    Ambiente de execução para rodar o projeto e scripts.
  </li>
  <li align="justify">
    <a href="https://jestjs.io/" target="_blank"><strong>Jest</strong></a>.
    Testes automatizados.
  </li>
  <li align="justify">
    <a href="https://eslint.org/" target="_blank"><strong>ESLint</strong></a>.
    Padronização e qualidade de código.
  </li>
  <li align="justify">
    <a href="https://www.docker.com/" target="_blank"><strong>Docker</strong></a>.
    Execução opcional via containers, quando disponível no projeto.
  </li>
</ul>

<h2><strong>Estrutura do Projeto</strong></h2>
<p align="justify">A estrutura do projeto segue a organização abaixo.</p>

<pre><code>
/
├─ src/                 # Código fonte do exercício
├─ tests/               # Testes automatizados
├─ images/              # Imagens para documentação (se aplicável)
├─ docker-compose.yml
├─ jest.config.js
├─ tsconfig.json
├─ .eslintrc.json
└─ package.json
</code></pre>
